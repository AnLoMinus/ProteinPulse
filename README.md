# 🧱 מבנה החלבון

### 🌀 1. חומצות אמינו – היסודות

חלבון מורכב מ־20 חומצות אמינו שונות.
כל חומצה אמינית היא כמו "אות", וכשהן מתחברות נוצרים “מילים” ו”משפטים” ביולוגיים.

### 🧩 2. סדר = משמעות

הסדר המדויק שבו החומצות האמיניות מתחברות משנה לחלוטין את תפקיד החלבון.
כמו סיפור שנכתב באותם אותיות אך בסדר אחר.

### 🎛️ 3. קיפול תלת־ממדי

החלבון מתקפל לצורה מדויקת כדי לבצע פעולה מסוימת:

* אנזים – צורה שמתאימה “מנעול־מפתח”.
* הורמון – צורה שמתחברת לקולטן מתאים.
* שריר – סיבים שמסוגלים להתכווץ ולהימתח.
